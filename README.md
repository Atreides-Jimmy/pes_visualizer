# PES Visualizer — 势能面搜索算法交互式可视化教学工具

## 项目简介

本项目是"人工智能化学计算模拟"课程大作业，实现了一个基于 PyQt5 + PyQtGraph 的势能面（Potential Energy Surface, PES）搜索算法交互式可视化教学工具。支持 2D 等高线与 3D 曲面双视角实时渲染，涵盖 18 种搜索算法与 8 种势能面类型，可逐步观察算法迭代过程、偏置势对势能面的动态修改、群体算法的种群分布等。

## 功能特性

### 势能面类型（8种）
| 势能面 | 公式 | 特点 |
|--------|------|------|
| Müller-Brown | E = Σᵢ Aᵢ·exp(aᵢ(x-x₀ᵢ)² + bᵢ(x-x₀ᵢ)(y-y₀ᵢ) + cᵢ(y-y₀ᵢ)²) | 经典测试函数，3极小值2鞍点 |
| 双阱 | E = (x²-1)² + y² | 两个极小值 |
| 三极小值 | E = x⁴/4 - x²/2 + y²/2 | 三个极小值 |
| Rosenbrock | E = (1-x)² + 100(y-x²)² | 狭长弯曲谷底 |
| Himmelblau | E = (x²+y-11)² + (x+y²-7)² | 四个等价极小值 |
| Ackley | E = -20·exp(-0.2·√(0.5(x²+y²))) - exp(0.5(cos2πx+cos2πy)) + e + 20 | 大量局部极小值 |
| Rastrigin | E = 20 + x²-10cos(2πx) + y²-10cos(2πy) | 规则分布的局部极小值 |
| 自定义 | 用户输入任意公式 | 支持^、2sin等简写 |

### 搜索算法（18种）

**极小值搜索：**
- 牛顿法(极小值) — q_{t+1} = q_t + H⁻¹·F
- 最速下降法 — q_{t+1} = q_t - α·∇E
- 盆地跳跃法 — 随机扰动+局部优化+Metropolis
- 遗传算法 — 选择+交叉+变异+局部优化
- 粒子群优化(PSO) — 惯性+认知+社会
- 人工蜂群(ABC) — 雇佣蜂+观察蜂+侦察蜂

**过渡态搜索：**
- 牛顿法(鞍点) — 反转Hessian最小本征值
- Dimer方法 — 双子数值求最低曲率方向
- CBD方法 — 约束布罗伊登双子法
- NEB方法 — 弹性带方法
- CI-NEB方法 — 攀爬弹性带方法
- DESW方法 — 双端行走法

**增强采样/全局搜索：**
- Metropolis MC — Boltzmann随机采样
- SSW方法 — 随机势能面行走（高斯偏置势可视化）
- 元动力学 — 高斯排斥势填井（偏置势可视化）
- 极小点跳跃 — MD逃逸+局部优化
- 伞形采样 — 谐振偏置窗口（偏置势可视化）
- 自适应偏置力(ABF) — 反向偏置力平坦化（偏置势可视化）

### 可视化特性
- **2D等高线**：彩色填充等高线 + 轨迹路径 + 起终点标记 + 种群散点
- **3D曲面**：OpenGL渲染 + 轨迹线 + 种群3D散点 + 可旋转/缩放
- **偏置势可视化**：SSW/元动力学/伞形采样/ABF/DESW 搜索过程中实时显示偏置势对势能面的修改，搜索结束后自动恢复
- **群体可视化**：GA/PSO/ABC 的种群/粒子位置以散点实时显示
- **信息面板**：实时物理量、势能面公式、算法原理与公式、关键术语、结果统计
- **算法对比**：支持同时运行两种算法进行对比
- **GIF导出**：三视图（等高线+两个3D视角）导出搜索过程动画

## 使用说明

1. **选择势能面**：左侧面板下拉选择势能面类型，自定义类型可输入公式
2. **设置初始点**：点击"设置初始点"按钮后在势能面上点击，或直接输入坐标
3. **选择算法**：选择搜索算法，调整参数后点击"确认参数"
4. **运行**：点击播放按钮开始迭代，可单步或自动播放
5. **观察**：2D/3D视图实时显示搜索轨迹，信息面板显示物理量和公式
6. **偏置势**：SSW/元动力学/伞形采样/ABF/DESW 运行时势能面会随偏置势动态变化
7. **导出**：可导出GIF动画记录搜索过程

## 技术要点

- **PyQtGraph 渲染**：2D 使用 `PlotWidget` + `ImageItem`，3D 使用 `GLViewWidget` + `GLSurfacePlotItem`
- **偏置势机制**：`set_bias_function()` 将偏置势叠加到原始势能面 `Z = Z_original + bias_grid`，收敛后清除
- **自定义公式解析**：支持 `^` → `**`、`2sin` → `2*sin`、`)(` → `)*(` 等简写
- **参数持久化**：算法参数在切换势能面/初始点时保留，仅切换算法时重置
- **GIF三视图**：等高线 + 两个3D视角水平拼接，使用 `QBuffer` 解决 `QImage.save()` 兼容问题
